Publishes your wordpress blog posts automatically to StockTwits

== Description == 

Publishes your wordpress blog posts automatically to StockTwits, using the StockTwits public API.  Add a custom messages with each post, automatically finds cashtags with in your post. 

== Installation ==

1. Install Publish to StockTwits onto your blog.
2. Go to Setting and select Publish to StockTwits.
3. Follow the instruction to obtain a consumer key for the StockTwits API.
4. Add your consumer key, and click update.
5. Add your StockTwits account by clicking the "Connect to StockTwits" button.
6. confirm the permissions for the app.
